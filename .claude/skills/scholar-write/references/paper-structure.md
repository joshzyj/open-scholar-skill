# Paper Structure Templates by Journal

## ASR / AJS Full Paper Template

### Title Page
```
[Title: 12 words or fewer; descriptive of phenomenon]
[Running head: ≤50 characters]
[Authors and affiliations — separate page for blind review]
[Acknowledgments — separate page for blind review]
[Word count]
[Correspondence: name, address, email]
```

### Abstract (150–200 words)
```
[Sentence 1–2: State the phenomenon and why it matters]
[Sentence 3–4: Describe data and method]
[Sentence 5–6: State main findings]
[Sentence 7: State contribution]
```

### Main Text Structure
```
Introduction (~800–1,000 words)
  - Hook
  - Phenomenon statement
  - Significance
  - Gap
  - What this paper does
  - Contribution
  - Roadmap

Background/Prior Research (~1,000–1,500 words)
  - Theoretical traditions
  - Empirical findings
  - Research gaps

Theory/Conceptual Framework (~800–1,200 words)
  - Primary theoretical argument
  - Mechanism
  - Hypothesis 1
  - Secondary argument / moderation
  - Hypothesis 2 (optional: H3)

Data and Methods (~1,000–2,000 words)
  - Data source
  - Sample construction
  - Dependent variable
  - Independent variable(s)
  - Controls
  - Analytic strategy

Results (~1,500–2,500 words)
  - Descriptive statistics paragraph
  - Model 1 paragraph
  - Model 2 paragraph (+ mediator or fuller controls)
  - Interaction/moderation paragraph
  - Robustness check paragraph

Discussion and Conclusion (~2,000–3,500 words)
  - Summary of findings
  - Theoretical interpretation/contribution
  - Comparison to prior literature
  - Scope conditions and limitations
  - Future directions
  - Conclusion

Notes (endnotes; minimal)
References
Tables (numbered; each on separate page)
Figures (numbered; each on separate page with caption)
```

**Total word limit**: ~10,000–12,000 (ASR); ~10,000–15,000 (AJS)

> **Empirical calibration**: For per-paper word counts from 53+ published papers, see `assets/article-knowledge-base.md` → "Empirical Section Word Counts by Journal."

## Demography Full Paper Template

### Structure
```
Title Page
Abstract (150 words) + Keywords
Introduction (~600–800 words)
Background (~1,000–1,500 words)
Data and Methods (~1,500–2,500 words)
  - Data
  - Measures (detailed)
  - Statistical approach
Results (~2,000–3,000 words)
  - Descriptive results (Table 1)
  - Main models (Tables 2–3)
  - Supplementary models
Discussion (~600–1,000 words)
Conclusion (~200–400 words)
References
Tables (numbered)
Figures
Online Supplementary Appendix
  A. Additional methods
  B. Supplementary tables
  C. Supplementary figures
```

**Key difference from ASR/AJS**: Demography expects a highly detailed Methods section; reviewers are methodologists who will look for robustness

## Science Advances Full Paper Template

### Structure
```
Title (≤150 characters)
Abstract (unstructured, ~250 words + one-sentence teaser)
Introduction (~800–1,200 words)
  [No "prior literature" section — integrate into intro]
Results
  [Organized by finding, not by model]
  [Each subsection: one key finding or analysis]
Discussion (~600–1,000 words)
Materials and Methods
  [After Discussion; can be detailed]
  [Or in Supplementary Materials if very long]
Supplementary Materials
  [Tables, additional analyses, code details]
References and Notes [numbered]
Acknowledgments
Author Contributions [CRediT taxonomy]
Competing Interests
Data Availability Statement
Code Availability Statement
```

**Key differences**:
- No separate "Literature Review" — intro integrates background
- Results section organized by finding, not by model specification
- Detailed Methods can go after Discussion

## Nature Human Behaviour / Nature Computational Science Template

### Structure
```
Title (≤90 characters)
Abstract (≤150 words; 3 sentences: Background / Findings / Implications)
[Main text — ~3,000–5,000 words]
  Introduction (no heading; ~500–700 words)
  Results [subheaded; organized by finding]
    [Finding 1 heading]
    [Finding 2 heading]
    [Finding 3 heading]
  Discussion (~600–900 words)
  Methods (~500–1,500 words, after Discussion)
    [Participants / Data]
    [Measures / Variables]
    [Statistical analyses]
    [Reporting summary reference]
Extended Data (up to 10 items; numbered ED Fig. 1, ED Table 1)
Supplementary Information
References (max 50 for articles)
Acknowledgments
Author Contributions
Competing Interests
Additional Information
  [Data availability statement]
  [Code availability statement]
  [Link to peer review]
```

**Nature abstract format**:
```
Sentence 1 (Background): "Although [established knowledge], [gap or tension] remains unclear."
Sentence 2 (Findings): "Here we show/find/demonstrate that [main finding], using [data/method]."
Sentence 3 (Implications): "Our findings suggest/reveal [theoretical or practical implication]."
```

**Extended Data** vs. **Supplementary Information**:
- Extended Data: Directly supports main text claims; peer-reviewed; cited as "Extended Data Fig. 1"
- Supplementary Information: Additional robustness, methods details; not peer-reviewed

## Section-Level Templates

### Introduction Sentence Templates

**Opening hook options**:
```
Option A (statistic): "In [year], [striking fact about population/trend]."
Option B (paradox): "Despite [policy/belief], [outcome] persists."
Option C (contradiction): "While [theory A] predicts X, [empirical observation] suggests Y."
Option D (question): "Why do [phenomenon]?"
```

**Gap statement**:
```
"Despite considerable research on [topic], less attention has been paid to [specific gap]."
"What remains unclear is whether [specific question]."
"No study has examined [specific phenomenon/population/context]."
```

**Contribution statement**:
```
"This paper contributes to [literature] in [N] ways."
"We advance this literature by [specific contribution]."
"Our findings speak to [theoretical debate] by [how]."
```

---

### Discussion Paragraph Templates

**Summary**:
```
"Our central finding is that [restate main result in plain language]. [Magnitude statement with effect size]."
```

**Theoretical interpretation**:
```
"This finding is consistent with [theory], which predicts that [mechanism]. When [condition], [outcome] follows because [explanation]."
```

**Comparison to prior work**:
```
"Our results corroborate/extend/challenge [Author Year], who found [their finding]. [Explain why yours differs or deepens their work]."
```

**Limitations**:
```
"This study has several limitations. First, [limitation]. Although [why it doesn't fatally undermine results], future research should [solution]."
```

**Future directions**:
```
"Future studies should examine [specific question]. In particular, [specific data/design/population] would allow researchers to [specific test]."
```

**Closing sentence**:
```
"In sum, this paper demonstrates that [main finding], with implications for [broader theoretical/policy relevance]."
```

## Theory / Background Paragraph Templates

### Mechanism Elaboration (main argument paragraph)

```
[Topic sentence: name the mechanism directly]
[Theoretical grounding: cite the framework that predicts this mechanism]
[Elaboration: explain HOW the mechanism operates — what the process looks like step by step]
[Empirical anchor: cite prior evidence that the mechanism exists]
[Link to hypothesis: "This logic leads us to expect that..."]
```

**Fill-in template**:
> "The primary mechanism linking [X] to [Y] is [mechanism name]. [Theory] (Author Year) predicts that when [condition], [process] occurs because [reason]. This process systematically [advantages / disadvantages] [group] by [how]. Consistent with this argument, prior research finds that [evidence] (Author Year; Author Year). We therefore expect that [H1 directional statement]."

---

### Moderation / Interaction Paragraph

```
[State the conditional claim: "The effect of X on Y is not uniform across groups"]
[Name the moderator and explain its theoretical relevance]
[Explain WHY the effect should be stronger/weaker for group Z]
[State the moderating hypothesis H2]
```

**Fill-in template**:
> "We argue that the effect of [X] on [Y] varies by [moderator]. For [group A], [reason why effect is stronger/present]. For [group B], [reason why effect is weaker/absent]. This differential [exposure / susceptibility / resource access] arises because [mechanism]. We therefore expect that [H2: the X–Y association is stronger / weaker for group A than group B]."

---

### Rival Explanation Paragraph

```
[Name the rival mechanism]
[Explain why it predicts the same outcome]
[Explain how your design addresses or distinguishes it]
[Or: explain why you cannot rule it out and what this implies]
```

**Fill-in template**:
> "One rival explanation for [expected finding] is [rival mechanism]. If [rival process] operates, we would expect [same outcome pattern]. Our design addresses this concern by [approach — e.g., conditioning on X, using within-person variation, restricting to sample]. Specifically, [how the design blocks the rival path or how data distinguish between the two mechanisms]."

---

### Boundary Condition / Scope Paragraph

```
[State the scope condition: the claim holds when/where/for whom]
[Explain the theoretical reason for the boundary]
[Note implications for generalizability]
```

**Fill-in template**:
> "This argument is most applicable in [context / time period / population]. The mechanism depends on [condition], which is [more / less] prevalent in [context]. We therefore expect [H direction] primarily among [scope group] and [weaker / no] effect among [out-of-scope group]. [Note: cross-national or historical replications would clarify whether [condition] is the operative scope condition]."

---

## Methods Paragraph Templates by Design Type

### Data Source Paragraph

**Fill-in template**:
> "We draw on the [Study Name], [brief description of what it contains and who collected it]. Data were collected [time period] using [survey / administrative / digital trace] methods. [Sampling design: nationally representative probability sample / administrative enrollment / scraped from platform]. The analytic sample is restricted to [criteria], yielding a final sample of N = [X] [units of analysis]."

---

### Dependent Variable Paragraph

**Fill-in template**:
> "Our dependent variable is [name], measured as [operationalization]. [Scale or coding: 'coded 1 if ...; 0 otherwise' / 'ranges from X to Y']. [Descriptive: mean = X, SD = X, or X% of sample = 1]. This measure captures [concept] because [validity rationale]."

---

### Key Independent Variable Paragraph

**Fill-in template**:
> "Our primary independent variable is [name], operationalized as [operationalization]. [Source and measurement details]. Variation in [X] arises from [cross-sectional comparison / panel change / quasi-experimental assignment]. [Descriptive statistics if relevant]. [Validity or measurement note if needed]."

---

### Analytic Strategy Paragraph — OLS / Logit

**Fill-in template**:
> "We estimate a series of [OLS / logistic regression] models regressing [Y] on [X], progressively adding control variables. Model 1 is the bivariate association. Model 2 adds [control block A — sociodemographic]. Model 3 adds [control block B — contextual]. All models use [HC3 robust standard errors / standard errors clustered at the [unit] level] to account for [heteroskedasticity / within-cluster correlation]. Because raw log-odds are difficult to interpret substantively, we report average marginal effects (AME) for all logistic models (Arel-Bundock, Greifer, and Heiss 2024)."

---

### Analytic Strategy Paragraph — Fixed Effects / DiD

**Fill-in template**:
> "To address unobserved time-invariant confounders, we estimate [two-way fixed effects / difference-in-differences] models including [unit] and [time period] fixed effects. Identification relies on within-[unit] variation in [X] over time. The estimating equation is: Y_it = α_i + λ_t + βX_it + γ'C_it + ε_it. Standard errors are clustered at the [unit] level to account for serial correlation (Bertrand, Duflo, and Mullainathan 2004). We test the parallel trends assumption using an event-study specification that plots estimates for each period relative to treatment onset (see Figure [N])."

---

### Analytic Strategy Paragraph — Regression Discontinuity

**Fill-in template**:
> "We exploit discontinuous assignment at [cutoff] on [running variable] using a regression discontinuity (RD) design. Units just above and below the threshold are plausibly similar on unobserved characteristics, rendering assignment as-if random near the cutoff. We estimate a [sharp / fuzzy] RD using rdrobust (Calonico, Cattaneo, and Titiunik 2014) with MSE-optimal bandwidth selection and triangular kernel weights. We test for manipulation of the running variable using the rddensity test and verify that predetermined covariates are continuous at the threshold."

---

## Results Paragraph Templates by Model Type

### Descriptive Statistics Paragraph (Table 1)

**Fill-in template**:
> "Table 1 presents descriptive statistics for the analytic sample. [Outcome variable: mean = X, SD = X / X% of sample = 1]. [Key independent variable: X% are [treated / in group A] vs. X% in [group B]]. [Notable distributional feature if relevant: e.g., 'The distribution is right-skewed, consistent with prior work on earnings inequality']. Groups differ significantly on [key covariate] (p < .05), motivating its inclusion as a control."

---

### Main Effect Paragraph (OLS / Logit)

**Fill-in template**:
> "[Finding statement first — consistent with / contrary to H1]. Table 2 presents the full regression results. In Model 1, [X] is [positively / negatively] associated with [Y] (b = [X], SE = [X], p [< .001 / = .0XX]). This association [persists / attenuates to b = X] after adding controls in Model 2 (b = [X], SE = [X]), suggesting that [interpretation]. [For logit:] The average marginal effect indicates that [X] is associated with a [X] percentage point [increase / decrease] in P([Y]) (AME = [X], 95% CI [[lo], [hi]])."

---

### Fixed Effects / Panel Result Paragraph

**Fill-in template**:
> "Within-[unit] estimates from the fixed effects models [confirm / do not support] the observational pattern. Among [units] that experienced a change in [X], [outcome direction] (b = [X], SE = [X], p = [X]; Model 3, Table 2). This within-unit estimate is [comparable to / substantially smaller than] the cross-sectional estimate (b = [X] vs. b = [X]), consistent with [interpretation: e.g., 'modest confounding from stable unobserved characteristics' / 'the full cross-sectional association is attributable to selection']."

---

### Interaction / Moderation Paragraph

**Fill-in template**:
> "Model [N] tests whether the effect of [X] on [Y] differs by [moderator], as predicted by H2. Figure [N] plots predicted [Y] at different levels of [X] separately by [moderator]. Consistent with H2, the association is stronger for [group A] (b = [X], 95% CI [[lo], [hi]]) than for [group B] (b = [X], 95% CI [[lo], [hi]]); the interaction coefficient is [statistically significant / not significant] (b = [Δ], SE = [X], p = [X])."

---

### Robustness Paragraph

**Fill-in template**:
> "Several robustness checks confirm the stability of these findings (Appendix Table A1). First, excluding [group / outliers / observations] does not substantively change the main estimate (b = [X], SE = [X]). Second, using [alternative operationalization of X / Y] yields similar results (b = [X]). [Third check if applicable.] These results support the conclusion that [main finding] is not an artifact of [specific methodological concern]."

---

### Null Finding Paragraph

**Fill-in template**:
> "Contrary to H[N], we find no statistically significant association between [X] and [Y] (b = [X], SE = [X], p = [X]; 95% CI [[lo], [hi]]). This null finding is informative: the confidence interval excludes effects larger than [magnitude], which is [well below / comparable to] the minimum effect that prior theory suggests would be [theoretically meaningful / practically consequential]. One interpretation is that [alternative explanation for the null]."
